# Statistics Foundation Course

## Course Overview
This 10-week course provides essential statistical knowledge for data engineering and analytics. From basic descriptive statistics to hypothesis testing and regression, this course builds the foundation needed for data-driven decision making.

## Prerequisites
- [Mathematics Foundation Course](../mathematics/README.md) (recommended)
- Basic algebra and arithmetic

## Time Commitment
- Estimated: 5-10 hours per week
- Total course duration: 10 weeks

## Course Objectives
By the end of this course, you will:
- Understand descriptive and inferential statistics
- Apply probability theory to real-world problems
- Conduct hypothesis tests and interpret results
- Perform correlation and regression analysis
- Make data-driven decisions with confidence
- Prepare for advanced machine learning concepts
