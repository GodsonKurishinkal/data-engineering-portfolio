# Programming Fundamentals Course (Python)

## Course Overview
This 8-week course provides comprehensive Python programming skills essential for data engineering. From basic syntax to data processing libraries, this course builds the programming foundation needed for data pipeline development.

## Prerequisites
- Basic computer skills
- Logical thinking ability
- Recommended: [Mathematics Foundation Course](../mathematics/README.md)

## Time Commitment
- Estimated: 8-12 hours per week
- Total course duration: 8 weeks

## Course Objectives
By the end of this course, you will:
- Write efficient Python code
- Work with various data structures
- Implement functions and modules
- Handle files and exceptions
- Apply OOP principles
- Use NumPy and Pandas for data processing
- Build foundation for data engineering projects

## Development Environment
- Python 3.8 or higher
- IDE: VS Code, PyCharm, or Jupyter Notebook
- Git for version control
- Virtual environments (venv or conda)
