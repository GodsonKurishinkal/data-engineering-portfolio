# Database Fundamentals Course

## Course Overview
This 8-week course provides comprehensive database knowledge essential for data engineering. From SQL basics to NoSQL databases, this course builds the data storage and retrieval foundation needed for modern data systems.

## Prerequisites
- [Programming Fundamentals](../programming-fundamentals/README.md)
- Basic understanding of data structures
- Command line familiarity

## Time Commitment
- Estimated: 8-12 hours per week
- Total course duration: 8 weeks

## Course Objectives
By the end of this course, you will:
- Master SQL for data retrieval and manipulation
- Design normalized database schemas
- Optimize queries for performance
- Understand NoSQL databases
- Choose appropriate database solutions
- Build foundation for data engineering pipelines
- Work with both relational and non-relational databases

## Database Systems Covered
- **Relational**: PostgreSQL, MySQL, SQLite
- **Document**: MongoDB
- **Key-Value**: Redis (introduction)
- **Concepts**: Cassandra, Neo4j

## Development Environment
- PostgreSQL 12+ or MySQL 8+
- SQLite for quick practice
- MongoDB Community Edition
- Database client tools (DBeaver, pgAdmin, MongoDB Compass)
- SQL editors and formatters
