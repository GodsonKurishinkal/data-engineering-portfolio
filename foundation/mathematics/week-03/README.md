# Week 3: Functions and Relations

## Learning Objectives
- Understand the concept of functions
- Identify domain and range
- Work with function notation
- Evaluate and compose functions

## Topics Covered

### 1. Functions Fundamentals
- Definition of a function
- Function notation f(x)
- Domain and range
- Vertical line test

### 2. Types of Functions
- Linear functions
- Quadratic functions
- Polynomial functions
- Rational functions

### 3. Function Operations
- Function composition
- Inverse functions
- Piecewise functions
- Transformations of functions

## Practice Problems

1. Given f(x) = 2x + 3, find f(5)
2. Find the domain and range of f(x) = √(x - 4)
3. If f(x) = x² and g(x) = x + 1, find (f ∘ g)(x)
