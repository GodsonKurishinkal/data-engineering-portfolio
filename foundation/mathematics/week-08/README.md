# Week 8: Matrices and Linear Algebra Basics

## Learning Objectives
- Perform matrix operations
- Understand matrix transformations
- Calculate determinants and inverses
- Apply matrices to data structures

## Topics Covered

### 1. Matrix Fundamentals
- Matrix notation and terminology
- Matrix addition and subtraction
- Scalar multiplication
- Matrix multiplication

### 2. Special Matrices
- Identity matrix
- Zero matrix
- Transpose matrix
- Symmetric matrices

### 3. Matrix Operations
- Determinants
- Matrix inverses
- Solving systems using matrices
- Row operations and row echelon form

## Practice Problems

1. Given A = [[1, 2], [3, 4]] and B = [[5, 6], [7, 8]], find A × B
2. Find the determinant of [[2, 3], [4, 5]]
3. Find the inverse of [[1, 2], [3, 4]]
