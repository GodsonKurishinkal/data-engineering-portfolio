# Week 12: Integration and Applications

## Learning Objectives
- Understand the concept of integration
- Calculate definite and indefinite integrals
- Apply the fundamental theorem of calculus
- Use integration in data analysis

## Topics Covered

### 1. Integration Fundamentals
- Antiderivatives
- Indefinite integrals
- Definite integrals
- Area under curves

### 2. Integration Techniques
- Power rule for integration
- Substitution method
- Integration by parts (introduction)
- Numerical integration

### 3. Applications
- Area and volume calculations
- Probability distributions
- Cumulative distribution functions
- Data aggregation and summation

## Practice Problems

1. Find the indefinite integral: ∫(3x² + 2x) dx
2. Evaluate: ∫₀² (x² + 1) dx
3. Find the area between y = x² and y = 4 from x = 0 to x = 2
