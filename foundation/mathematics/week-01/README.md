# Week 1: Introduction to Mathematical Foundations

## Learning Objectives
- Understand basic mathematical notation and terminology
- Review fundamental arithmetic operations
- Introduction to algebraic expressions
- Sets and basic set theory

## Topics Covered

### 1. Mathematical Notation
- Variables and constants
- Operators: +, -, ×, ÷
- Exponents and roots
- Order of operations (PEMDAS/BODMAS)

### 2. Algebraic Fundamentals
- Simplifying expressions
- Combining like terms
- Distributive property
- Factoring basics

### 3. Sets and Set Theory
- Set notation and representation
- Set operations (union, intersection, difference)
- Venn diagrams
- Cardinality

## Practice Problems

1. Simplify: 3(x + 4) - 2(x - 1)
2. Evaluate: 2³ + 4² - √16
3. Given sets A = {1, 2, 3, 4} and B = {3, 4, 5, 6}, find A ∪ B and A ∩ B
