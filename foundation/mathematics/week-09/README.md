# Week 9: Vectors and Vector Operations

## Learning Objectives
- Understand vector concepts and notation
- Perform vector operations
- Calculate dot and cross products
- Apply vectors to data science problems

## Topics Covered

### 1. Vector Fundamentals
- Vector notation and representation
- Vector addition and subtraction
- Scalar multiplication
- Unit vectors and normalization

### 2. Vector Operations
- Dot product (inner product)
- Cross product
- Vector magnitude and direction
- Angle between vectors

### 3. Applications
- Vector spaces in data science
- Feature vectors
- Distance metrics
- Similarity measures

## Practice Problems

1. Given u = [3, 4] and v = [1, 2], find u + v and ||u||
2. Calculate the dot product of [1, 2, 3] and [4, 5, 6]
3. Find the angle between vectors [1, 0] and [1, 1]
