# Week 2: Linear Equations and Inequalities

## Learning Objectives
- Solve linear equations in one variable
- Work with linear inequalities
- Graph solutions on number lines
- Apply linear equations to real-world problems

## Topics Covered

### 1. Linear Equations
- One-variable linear equations
- Two-step and multi-step equations
- Equations with variables on both sides
- Literal equations

### 2. Linear Inequalities
- Inequality symbols (<, >, ≤, ≥)
- Solving simple inequalities
- Compound inequalities
- Graphing solutions

### 3. Applications
- Word problems involving linear equations
- Rate, time, and distance problems
- Mixture problems
- Age problems

## Practice Problems

1. Solve for x: 3x - 7 = 2x + 5
2. Solve and graph: 2x + 3 < 11
3. Solve: 5(x - 2) = 3(x + 4)
