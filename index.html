What I do

Core specialisations.

End-to-end data engineering for enterprise supply chain operations.

01

Medallion lakehouse

Bronze → Silver → Gold on Hive-partitioned Parquet. Designed for lineage, incremental loads, and 30,000+ SKUs flowing daily.

02

ETL frameworks

Configuration-driven pipelines built on Abstract Base Classes. 50+ production pipelines, 95%+ reliability, FULL / INCREMENTAL modes.

03

Dimensional modelling

Kimball star schema with 10+ fact tables, 6+ conformed dimensions, SCD Type 2. Optimised for analytical queries and semantic layers.

04

Data quality

3-tier validation: schema → business rules → statistical outliers. 500+ issues caught before reaching Gold; incidents down 70%.

05

ML forecasting

7+ algorithm ensemble with ADI/CV² pattern classification. Prophet, XGBoost, Croston — chosen per SKU, not per portfolio.

06

RPA & legacy integration

Selenium + PyAutoGUI bots for systems with no APIs (WMS, OBI, ERP, POS). When there's no way in, I build one.
